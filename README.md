# basic-player
A web player that supports video and audio

![](https://i.ex3.lol/UuCSeTT8Yy)
## What is this for?
This could be useful for:
- Precise playback speed control
- For sites that force you to download a file instead of viewing it
- If you want a basic visualizer
## Why this may not work
- CORS
- You might just simply not need it
